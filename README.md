# Vue 3 + TypeScript + Vite + m


## **安装依赖**
### 安装全局依赖
```shell
pnpm i -w
```

### 新增全局依赖
```shell
pnpm i 包名 -w
```

### 新增局部依赖
```shell
pnpm i 新增的包名 -F 安装到指定到子包(包名)
```

### 子包相互依赖

```shell
pnpm i 待安装到子包 -F 子包
```
